# Elevatri — Site

Web landing for BookSlide by Elevatri.

Git support added locally.

Cómo inicializar y conectar a un remoto:

```bash
cd /Users/cborgesm1pro/Documents/Elevatri/Site
# inicializar repo y rama main
git init -b main
# configurar nombre/email local (opcional)
git config user.name "Tu Nombre"
git config user.email "tu@correo"
# añadir archivos y commit
git add .
git commit -m "Initial commit"
# añadir remoto y push (ejemplo)
git remote add origin git@github.com:tuusuario/tu-repo.git
git push -u origin main
```

Notas:
- `.gitignore` incluye patrones comunes para proyectos web.
- Si prefieres que yo inicialice el repo y haga el commit, dime y lo hago (necesitaré permiso para ejecutar comandos en tu terminal).
